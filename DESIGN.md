# DESIGN.md

## 气质与意象

夏日午后的小学课桌，阳光透过纱窗洒在绿色的AD钙奶瓶上，瓶身挂着冷凝水珠。手指捏住瓶盖一拧，"啵"的一声，气泡从瓶口涌出，世界在那一刻回到六岁。整个产品就是这声"啵"的延长——一种被打开的快乐，不分年龄。

## 配色方案

- **AD钙奶绿** `#4CAF50`：瓶身标志性绿，所有主CTA、标题强调、进度条主色。意象：握在手里的绿色瓶子
- **奶白色** `#FFF8E1`：全局背景基调。意象：牛奶倒在杯中
- **活力橙** `#FF9800`：选中态、高亮、庆祝撒花。意象：吸管末端那抹橙色
- **深绿** `#2E7D32`：像素风元素、边框描边。意象：复古标签上的深色印刷
- **柔黄** `#FFF9C4`：星光装饰、结果卡高光。意象：阳光透过瓶子折射的光斑
- **浅粉** `#FFCDD2`：少女心选项反馈。意象：童年贴纸

## 字体排版

- 标题：系统圆体栈 `"PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", sans-serif`，加粗 + letter-spacing 放宽，模拟手写广告牌质感
- 正文：同一字体栈，regular weight
- 像素风段落：`"Press Start 2P", monospace`（英文像素字体，中文用系统等宽体 fallback）
- 排版节奏：大标题 28px / 小标题 20px / 正文 16px / 辅助 13px，行高 1.6，段间距 12px

## 视觉策略

- 所有图标和插画用 **SVG 内联** 或 **CSS 绘制**，不依赖外部图片
- 背景永远有浮动气泡 + 闪烁星点，营造"瓶中世界"氛围
- 答题反馈用卡通表情 + 弹跳动画，选C给"瓶盖喷射"级特效
- 五种挑战视觉风格用 CSS class 切换，不换资源：像素（像素化边框+等宽字体）→ 手绘（波浪边框+倾斜）→ 3D（投影+高光）→ 漫画（粗描边+网点）→ 霓虹（发光+暗底）

## 动效与交互

- 运动性格：**弹跳、弹簧感**，ease-out-back 曲线，不使用线性或ease-in
- 页面切换：AD钙奶瓶居中放大 → 瓶盖弹飞 → 气泡涌出 → 新内容从气泡中浮现
- 选项点击：scale(0.9) → scale(1.05) → scale(1)，200ms
- 倒计时数字：flip 动画翻牌
- 挑战成功：绿色气泡爆发 + ✨
- 挑战失败：瓶子轻微晃动 + 搞笑文字飘入
- 气泡：从底部上浮，大小随机 8-24px，透明度 0.15-0.4，速度 4-8s，无限循环

## 页面结构

- 所有页面 **垂直居中**，375px 基准宽度，最大 430px
- 顶部留状态栏空间（safe-area-inset-top）
- 底部CTA按钮固定在可视区域下方，距离底部 safe-area
- 气泡层 z-index: 0，内容层 z-index: 1，弹窗/动画层 z-index: 10

## 设计禁忌

- ❌ 不要用冷色系蓝紫渐变（与AD钙奶温暖调性冲突）
- ❌ 不要用直角/锐角元素（全部圆角 ≥ 12px）
- ❌ 不要用系统默认 alert/prompt（全部自定义弹窗）
- ❌ 不要用纯白 #FFF 背景（用奶白 #FFF8E1）
- ❌ 不要用线性 ease 动画（全部用弹性曲线）
- ❌ 不要出现空白无装饰区域（至少有气泡/星点）
