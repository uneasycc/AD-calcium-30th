# 项目上下文

## 项目简介

AD钙奶30周年线上互动H5网页应用，主题「不管几岁 快乐万岁」，移动端优先适配。

## 技术栈

- **核心**: Vite 7, TypeScript, Express
- **UI**: Tailwind CSS 3 + 自定义CSS动画
- **图片生成**: html2canvas
- **音频**: Web Audio API（合成音效，无外部音频文件）

## 目录结构

```
├── scripts/            # 构建与启动脚本
├── server/             # Express 服务端（静态文件服务 + API）
├── src/
│   ├── index.css       # 全局样式、动画、主题变量
│   ├── index.ts        # 入口文件
│   ├── main.ts         # 应用引导
│   ├── app.ts          # 主状态机 & 阶段切换
│   ├── types.ts        # 共享类型定义
│   ├── sound.ts        # Web Audio API 音效
│   ├── bottle.ts       # 瓶子动画 & SVG组件
│   ├── quiz.ts         # 环节一：快乐年龄测试仪
│   ├── challenge.ts    # 环节二：30秒快乐挑战
│   └── profile.ts      # 环节三：快乐档案生成
├── index.html          # 入口 HTML
├── DESIGN.md           # 设计规范文件
└── AGENTS.md           # 本文件
```

## 包管理规范

**仅允许使用 pnpm** 作为包管理器，**严禁使用 npm 或 yarn**。

## 开发规范

- 使用 Tailwind CSS + CSS 自定义属性进行样式开发
- 所有视觉元素使用 SVG/CSS 绘制或 emoji 替代，不依赖外部图片
- 音效使用 Web Audio API 合成，不依赖外部音频文件
- 移动端优先（375px 基准宽度），兼容主流手机浏览器
- 触控手势交互（点击、滑动、长按），桌面端有鼠标降级方案

### 编码规范

- TypeScript strict 模式，禁止隐式 any
- 函数参数和返回值需要明确类型
- 清理未使用的变量和导入
- 事件监听器在组件销毁时需清理（特别是 document 级别的监听器）

## 应用流程

1. **开屏页** → 点击AD钙奶瓶盖 → 开瓶动画
2. **环节一：快乐年龄测试仪** → 5道趣味答题 → 快乐年龄结果
3. **环节二：30秒快乐挑战** → 10个快闪小互动 → 快乐等级
4. **环节三：快乐档案生成** → 档案卡 + 保存/分享 + 彩蛋
5. 每个环节切换都伴随开瓶动画过渡
